# 出力フォーマット集 — 共有スキル

KCP全体で標準として使う出力テンプレートのライブラリ。

---

## 意思決定サマリ（どのAgentでも使える）

```markdown
## 意思決定サマリ

**決定事項**: [1文で]
**根拠**: [2〜3文]
**リスク**: [主要リスク1点]
**次のアクション**: [期日付き]
```

---

## タスク定義（Operatorが使う）

```markdown
## タスク: [タイトル]
- **担当**: [Chief/Specialist]
- **期日**: YYYY-MM-DD
- **優先度**: 高/中/低
- **完了基準**: [何ができたら完了か]
- **依存関係**: [これが完了しないと進められないタスク]
```

---

## 会議メモ（汎用）

```markdown
## 会議メモ
日時: YYYY-MM-DD HH:MM
参加者: [名前]
議題: [トピック]

### 議論内容
[主要な議論のポイント]

### 決定事項
- [決定1]
- [決定2]

### アクションアイテム
| アクション | 担当 | 期日 |
|-----------|------|------|
| [アクション] | [担当] | YYYY-MM-DD |
```

---

## プロジェクトステータス更新

```markdown
## [プロジェクト名] ステータス更新
更新日: YYYY-MM-DD

| 項目 | 内容 |
|------|------|
| 全体進捗 | X% |
| 現在のフェーズ | [フェーズ名] |
| 次のマイルストーン | YYYY-MM-DD |
| ブロッカー | [あれば記載] |
| ステータス | 🟢 順調 / 🟡 要注意 / 🔴 遅延 |
```
