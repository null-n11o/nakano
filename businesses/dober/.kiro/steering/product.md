# Product Overview

Doberは20〜30代男性向けの個人ブランド × コンテンツビジネス。
「精神・恋愛・経済・知力・身体」の5領域で発信し、Threadsを入口とするファネルで収益を生む自社プロダクト事業。

## Core Capabilities

1. **コンテンツ発信** — Threads投稿（n8n自動化済み）で毎日フックを打ち、エンゲージメントを集める
2. **リードマグネット配布** — キーワードリプライをトリガーに、カテゴリ別LMをDMで自動配布してメールリストへ誘導
3. **有料商品販売** — 恋愛OS・金融OS・規律プロトコルのバンドルをメールリスト経由で販売

## Target Use Cases

- 「評価」「資産」「規律」とリプライした人がLMを受け取り、メールリストにリストインする
- リスト登録者が有料商品（恋愛OS + 金融OS バンドル）を購入する

## Value Proposition

**ブランドトーン**: 断定調・命令調。感情論排除、数字と観察事実のみ。コンサル・軍隊的な冷徹さ。
「感情ではなくシステムで動く男」へのOS提供がDoberの核心的価値。

## Current Phase

**Phase 1: ファネル拡張**（現在地）
- LM-A（GRM評価シート）作成 → n8n DM分岐実装 → リスト64→300人

**Phase 2以降**: 有料商品作成・セール → スケール
