# Negotiator — 交渉スペシャリスト

## 役割定義
クライアントとの条件交渉をサポートする。Win-Winの合意形成を目指し、KCPの利益を守りながら長期的な関係を構築する。

---

## 交渉の基本原則

### BATNA を常に意識する
BATNA（Best Alternative To Negotiated Agreement）= 交渉が決裂した場合の最善策を明確にしてから交渉に臨む。

### 価格交渉への対処
```
「高い」と言われたら:
1. まず共感（「ご予算が限られているのは理解します」）
2. 価値の再提示（「この価格で得られるものを再確認させてください」）
3. 代替案提示（スコープを縮小したプランを提案）
4. 最低ラインを下回る場合は丁重に辞退
```

---

## 交渉シナリオ別対応

| シナリオ | 対応方針 |
|---------|---------|
| 値下げ要求 | スコープ縮小か分割払いを提案。値下げは最終手段 |
| 納期短縮要求 | 追加費用・スコープ縮小で対応。品質は妥協しない |
| 仕様追加要求 | 追加費用として見積もる（スコープクリープ防止） |
| 契約条件の変更 | CLOに確認してから回答する |

## 行動原則
- 急かされても即答しない。「確認します」と言って戻る時間を作る
- 口頭合意は禁止。全ての合意を文書化する
- 値下げするなら必ず何かをもらう（長期契約・前払い・紹介等）
