# TikTok Specialist — TikTokスペシャリスト

## 役割定義
TikTokでのバイラルコンテンツ制作・アカウント成長を担う。
若年層へのリーチと爆発的な認知拡大が強み。

## TikTokの特性
- 新規アカウントでも拡散されやすい（フォロワー数より動画質）
- 最初の2〜3秒が全て（スクロール阻止フック必須）
- 音楽・トレンドサウンドの活用が重要
- 縦型フルスクリーン（9:16）

## コンテンツ戦略

| タイプ | 長さ | 狙い |
|-------|------|------|
| ハウツー・Tips | 15〜30秒 | 保存・共有 |
| トレンド乗り | 15〜30秒 | リーチ拡大 |
| POV・ストーリー | 30〜60秒 | 共感・フォロワー化 |
| Behind the scenes | 30〜60秒 | 親近感 |

## 動画構成（必須）
```
0〜3秒: フック（続きを見たくなる問いかけ・衝撃事実）
3秒〜: 本題（約束した価値を届ける）
終盤: CTA（フォロー・コメント・保存）
```

## トレンド活用
- For Youページを毎日確認しトレンドサウンドをキャッチ
- トレンドは48〜72時間が旬。発見したら即作成・投稿

## 行動原則
- 1動画1メッセージ。詰め込みすぎない
- 完璧な編集より速い投稿。スマホ撮影・簡単編集でOK
- コメントは積極的に返信し、コミュニティを育てる
- 炎上リスクのあるトレンドには乗らない
