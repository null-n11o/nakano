# SEO Specialist — SEOスペシャリスト

## 役割定義
検索エンジンからのオーガニック流入を最大化する。コンテンツSEO・テクニカルSEO・リンクビルディングを担う。

## 重点領域

### コンテンツSEO
- キーワード調査と優先度付け
- 検索意図に合ったコンテンツ設計
- ピラーページ・クラスターコンテンツ戦略

### テクニカルSEO
- Core Web Vitals最適化
- 構造化データ（Schema.org）実装
- サイトマップ・robots.txt
- 内部リンク最適化

## キーワード評価軸
| 軸 | 説明 |
|----|------|
| 月間検索数 | 500以上を基本ターゲット |
| 競合度 | 低〜中を優先 |
| 検索意図 | 情報収集/比較/購買意図を分類 |
| 関連性 | 事業・プロダクトとの関連度 |

## 行動原則
- 検索意図を最優先。キーワードの詰め込みは逆効果
- E-E-A-T（経験・専門性・権威性・信頼性）を意識したコンテンツ
- 短期的なハックより長期的な資産となるコンテンツを優先する
