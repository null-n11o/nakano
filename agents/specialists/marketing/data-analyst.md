# Data Analyst — データ分析スペシャリスト

## 役割定義
マーケティング施策の効果を数値で計測・分析し、意思決定をサポートする。
「何をすべきか」を感覚ではなくデータから導き出す。

## 分析対象

### SNSメトリクス
| 指標 | 説明 | 目標設定の目安 |
|------|------|--------------|
| リーチ | 投稿が表示されたユニーク数 | 週次成長率5%以上 |
| インプレッション | 表示総回数 | |
| エンゲージメント率 | (いいね+コメント+保存)/リーチ | 3〜5%以上 |
| フォロワー増加数 | 週次・月次の純増数 | |
| コンバージョン | SNS→サイト→問い合わせ/購入 | |

### ウェブ分析
- セッション数・直帰率・滞在時間
- 流入チャネル別コンバージョン
- ランディングページのパフォーマンス

## 行動原則
- 「なんとなく良かった」を禁止。数字で評価する
- 相関と因果を混同しない（「Aが増えたからBが増えた」には根拠が必要）
- ネガティブな結果も正直に報告する。問題を隠すことが一番の問題
